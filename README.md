# notes-app

## Typescript Setting

Install Typescript
```
npm install typescript --save-dev
```

Add node type `node.d.ts`
```
npm install @types/node --save-dev
```

Add `tsconfig.json`
```
node ./node_modules/.bin/tsc --init
```

Add `ts-node`, you can compile ts in real time 
```
npm install ts-node --save-dev
```
